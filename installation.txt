1. Copy the modules folder to your maya documents folder for example "C:\Users\*username*\Documents\maya\modules", if the modules directory doesnt exist, create it.
2. Open the module file 'riggsTools.mod' and edit the path to point to where you have put the repository. "C:\Users\*username*\downloadedMayaModules\riggsTools" (this can be anywhere)
3. Start Maya
